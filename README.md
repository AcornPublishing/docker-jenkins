# 도커와 젠킨스로하는 지속적 인도, 2판


이 책에서 다루는 내용:
* 도커의 기초를 배우고, 지속적 인도 프로세스에 맞도록 애플리케이션을 도커 이미지로 제작하
* 클라우드 환경에서 젠킨스를 활용하기
* 쿠버네티스로 도커 서버 풀을 확장하기
* 큐컴버로 인수 테스트를 작성하고, 이를 도커 환경에서 젠킨스로 실행하기
* 도커 레지스트리에 이미지를 발행하고, 모범 사례에 맞춰 젠킨스 파이프라인을 배포하기
